(2014.1 - Current)

这几周没什么大事，就翻出来好久没动的网站重新改一改。
应该是建站3年中的第三次改版了，真是每年必会改一改，每年看自己一年前写的网站（如头图）都会浑身难受（一年前写完的时候还是觉得很漂亮的)

更新日志：

  • 终于还是决定换成 Hexo 了
  • 重制所有项目（进行中）
  • Hexo 基于 edinburgh 主题自己又改了一些，这主题很多玩意都得自己写坑挺多

上一版的简介还是写的 junior student，翻了一下 GitHub 的记录，上一次 push 还是去年 4 月，那段时间为了找实习，每天能 push 十多次。大四一年算是实习了半年然后好好放松了半年（大学生活真好啊真好啊.jpg)，真是提前成为社畜就开始松懈啊。

该挖的坑早晚都要挖的，这个网站荒废着也不是个事儿。之前两版都是 PORTFOLIO 展示，本来想写成既有 PORTFOLIO 又有 BLOG 的，但是 Hexo 限制太多（其实是自己好久没写代码已经不太会写了），试了好多次也没能搞定。最后还是硬写了个 “伪·PORTFOLIO”，等着自己什么时候有时间再鼓捣。
